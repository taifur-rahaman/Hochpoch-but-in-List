# Salary Calculator

This is a simple Python program that calculates a user's weekly salary based on the number of hours worked each day.

## Features

* Prompts the user for their **name** and **ID**.
* Asks for working hours for each weekday (Monday to Friday).
* Calculates the total salary using the given hourly wage.
* Displays a summary including total hours and final payment.

## How It Works

The program loops through a list of weekdays, asking the user to input the hours worked on each day. Using a fixed hourly wage, it multiplies and sums up the total.
